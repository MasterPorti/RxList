# RxList 💊

> AI-Powered Prescription Management for Healthcare Professionals

RxList is a medical application designed for doctors and nurses to streamline prescription management using artificial intelligence.

## What is RxList?

RxList transforms the traditional prescription workflow:

1. **📷 Scan** — Upload or photograph the patient's Rx prescription list
2. **🤖 AI Processing** — Our AI engine reads, identifies, and validates each medication automatically
3. **✅ Smart Checklist** — Receive an organized checklist with dosages, schedules, and interaction alerts

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Nothing Phone-inspired design system)
- **Fonts**: Geist Sans & Geist Mono

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Design

Inspired by Nothing Phone's minimalist glyph interface:
- Monochrome dark palette with red accent
- Dot-matrix background patterns
- Geometric glyph animations
- Clean, monospaced typography elements
